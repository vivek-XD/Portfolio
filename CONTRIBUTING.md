# Contributing

Thanks for your interest! This is a personal portfolio, but suggestions are welcome.

## How to suggest changes
1. Fork the repo
2. Create a branch (`git checkout -b improve-something`)
3. Make your changes to `index.html`
4. Commit and open a Pull Request describing what you changed

## Reporting issues
Found a bug or broken link? Open an issue with:
- What you expected
- What actually happened
- Browser & device

Keep it simple, keep it kind. 🙌
